    **Data hooks** (frontend/src/hooks/):

    useFinancials.ts:
    - `useMetrics()`: TanStack Query hook calling GET /api/dashboard/metrics with selectedPropertyId in query key. Returns { ytd_revenue, ytd_expenses, current_month_profit, yoy_revenue_change, yoy_expense_change, actions_count }
    - `useOccupancy()`: TanStack Query hook calling GET /api/dashboard/occupancy with selectedPropertyId. Returns per-property monthly occupancy data.
    - Both hooks include selectedPropertyId from Zustand store in the query key so data refetches when property changes.
    - Set staleTime to 5 minutes (data doesn't change frequently).

    useActions.ts:
    - `useActions()`: TanStack Query hook calling GET /api/dashboard/actions with selectedPropertyId. Returns { items: ActionItem[], total_count: number }.
    - Export ActionItem type: { type, booking_id, guest_name, ... } (union type for the 3 action types).

    **StatCard** (frontend/src/components/home/StatCard.tsx):
    - Props: title, value (string), yoyChange (string | null), tooltipText (string)
    - Render: shadcn Card with large number display, title below
    - YoY change: green text with up arrow for positive, red with down arrow for negative, gray for null/zero
    - Tooltip on the title (using shadcn Tooltip) with plain-language explanation:

      - Expenses: "Total costs including rent, utilities, and maintenance this year"

    - Show SkeletonCard when data is loading

    - Bars colored by platform using the platform color palette (stacked bar chart: Airbnb red, VRBO blue, RVshare sky)

    - Loading: show Skeleton placeholder matching chart dimensions
