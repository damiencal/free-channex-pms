**Props:**
```typescript
interface MonthlyTableProps {
  months: string[]                    // ordered month keys e.g. ["2026-01", "2026-02", ...]
  rows: Array<{
    label: string                     // row label (account/category name)
    values: Record<string, string>    // month key -> amount string
    total: string                     // row total across all months

    isGrandTotal?: boolean            // grand total row styling
  }>
  showPercentage?: boolean            // show % column (P&L uses this)
  percentageBase?: string             // total to compute % against (revenue total)
}
```

**Month label formatting:**
- Helper function `formatMonthLabel(monthKey: string): string` — converts "2026-01" to "Jan 2026", "2026-12" to "Dec 2026". Use `new Date(year, monthIndex, 1).toLocaleDateString('en-US', { month: 'short', year: 'numeric' })`.

**Number formatting** (same helper as BalanceSheetTab — duplicate locally or define inline):
- `formatAmount(value: string)` returning { display, isNegative, isZero }
- Zero → em-dash "—"
- Negative → red text with parentheses
- Use `tabular-nums` class on all amount cells

**Layout (see 12-RESEARCH.md Pattern 6):**
```tsx
<div className="overflow-x-auto print:overflow-visible">
  <table className="w-full min-w-max text-sm">
    <thead>
      <tr className="border-b">
        <th className="sticky left-0 z-10 bg-card px-3 py-2 text-left font-medium min-w-[180px]">
          {/* empty or "Account" label */}
        </th>
        {months.map(m => (
          <th key={m} className="px-3 py-2 text-right font-medium whitespace-nowrap">
            {formatMonthLabel(m)}
          </th>
        ))}
        <th className="px-3 py-2 text-right font-medium whitespace-nowrap">Total</th>
        {showPercentage && <th className="px-3 py-2 text-right font-medium whitespace-nowrap">%</th>}
      </tr>
    </thead>
    <tbody>
      {rows.map((row, i) => ...)}
    </tbody>
  </table>
</div>
```

```typescript
function formatPercent(amount: string, base: string): string {
  const num = parseFloat(amount)
  const baseNum = parseFloat(base)
  if (isNaN(num) || isNaN(baseNum) || baseNum === 0 || num === 0) return '\u2014'
  return (Math.abs(num) / Math.abs(baseNum) * 100).toFixed(1) + '%'
}
```
