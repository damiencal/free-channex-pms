4. **Workflow: Generate Financial Reports** (curl examples):
   - P&L Report with various period options:
     ```bash
     # Year-to-date
     curl "http://localhost:8000/api/reports/pl?ytd=true"

     # Specific year
     curl "http://localhost:8000/api/reports/pl?year=2026"

     # Specific month
     curl "http://localhost:8000/api/reports/pl?month=3&year=2026"

     # Specific quarter
     curl "http://localhost:8000/api/reports/pl?quarter=Q1&year=2026"
     ```
   - Balance Sheet:
     ```bash
     curl "http://localhost:8000/api/reports/balance-sheet"
     ```
   - Income Statement:
     ```bash
     curl "http://localhost:8000/api/reports/income-statement?year=2026"
     ```

5. **Workflow: Natural Language Query** (curl example — SSE streaming):
   ```bash
   curl -X POST http://localhost:8000/api/query/ask \
     -H "Content-Type: application/json" \
     -d '{"message": "What was my total revenue last month?"}' \
     --no-buffer
   ```
   - Explain SSE response format: events arrive as `data: {...}` lines
   - Event types: `sql` (generated SQL query), `results` (query results), `token` (streaming narrative tokens), `done` (completion), `error` (if failed)
   - Note: Requires Ollama to be running. Returns error event if Ollama unavailable.
   - JavaScript example using EventSource or fetch with ReadableStream (brief)

6. **Workflow: Record a Loan Payment** (curl example):
   ```bash
   curl -X POST http://localhost:8000/api/accounting/loans/payments \
     -H "Content-Type: application/json" \
     -d '{
       "loan_id": 1,
       "principal": "500.00",
       "interest": "125.00",
       "payment_date": "2026-03-01",
       "payment_ref": "2026-03"
     }'
   ```
   - Explain: Creates balanced journal entries (debit loan liability + interest expense, credit bank asset)
   - `payment_ref` serves as idempotency key — prevents duplicate payments

7. **Workflow: Bank Reconciliation** (curl example):
   ```bash
   # Trigger auto-matching
   curl -X POST http://localhost:8000/api/accounting/reconciliation/run

   # View unreconciled items
   curl http://localhost:8000/api/accounting/reconciliation/unreconciled

   # Confirm a match
   curl -X POST http://localhost:8000/api/accounting/reconciliation/confirm \
     -H "Content-Type: application/json" \
     -d '{"match_id": 1}'
   ```
