- `saved: boolean` — flash green checkmark for 1.5s after successful save

Use `useMutation` from TanStack Query directly (import `categorizeTransaction` from `@/api/finance`). On success: set `saved=true`, `setTimeout(() => setSaved(false), 1500)`, invalidate `['finance']` queries, call `onSaved?.()`, clear `pendingCategory`.

Rendering logic:

Multi-select logic:
```typescript
const allIds = allTransactions.map(t => t.id)
const isAllSelected = allIds.length > 0 && allIds.every(id => selected.has(id))

function toggleRow(id: number) {
  setSelected(prev => {
    const next = new Set(prev)
    next.has(id) ? next.delete(id) : next.add(id)
    return next
  })
}

function toggleAll() {
  setSelected(isAllSelected ? new Set() : new Set(allIds))
}
```

Filter change handler: Reset offset to 0, clear selected set, clear accumulated transactions:
```typescript
function handleFilterChange(newFilters: TransactionFilters) {
  setFilters({ ...newFilters, limit: 50, offset: 0 })
  setSelected(new Set())
  setAllTransactions([])
}
```
