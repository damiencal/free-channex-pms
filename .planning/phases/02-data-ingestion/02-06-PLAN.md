<task type="auto">
  <name>Task 1: Create ingestion API endpoints</name>
  <files>app/api/ingestion.py</files>
  <action>
    Create app/api/ingestion.py with an APIRouter (prefix="/ingestion", tags=["ingestion"]) providing these endpoints:

    **1. POST /ingestion/airbnb/upload — Airbnb CSV upload**
    ```python
    @router.post("/airbnb/upload")
    async def upload_airbnb_csv(
        file: UploadFile = File(...),
        db: Session = Depends(get_db),
    ):
    ```
    - Validate file is .csv extension
    - Read bytes: `raw_bytes = await file.read()`
    - Call `normalizer.ingest_csv(raw_bytes, file.filename, "airbnb", airbnb_adapter, db)`
    - Catch ValueError -> return HTTPException 422 with error details
    - Return result dict (inserted, updated, skipped counts + IDs)

    **2. POST /ingestion/vrbo/upload — VRBO CSV upload**
    Same pattern as Airbnb but with vrbo adapter and platform="vrbo".

    **3. POST /ingestion/mercury/upload — Mercury CSV upload**
    ```python
    @router.post("/mercury/upload")
    async def upload_mercury_csv(
        file: UploadFile = File(...),
        db: Session = Depends(get_db),
    ):
    ```
    - Same validation and byte reading
    - Call `normalizer.ingest_bank_csv(raw_bytes, file.filename, mercury_adapter, db)`
    - Same error handling and response pattern

    **4. POST /ingestion/rvshare/entry — RVshare manual booking entry**
    ```python
    @router.post("/rvshare/entry")
    def create_rvshare_booking(
        entry: RVshareEntryRequest,
        db: Session = Depends(get_db),
    ):
    ```
    - Accept JSON body validated by Pydantic RVshareEntryRequest
    - Call `normalizer.create_manual_booking(entry, db)`
    - Catch ValueError -> return HTTPException 422 (e.g., invalid property_slug)
    - Return result dict

    **5. GET /ingestion/history — Import run history**
    ```python
    @router.get("/history")
    def get_import_history(
        platform: str | None = None,
        limit: int = 50,
        db: Session = Depends(get_db),
    ):
    ```
    - Query ImportRun table, ordered by imported_at descending
    - Optional platform filter (if provided, filter WHERE platform = platform)
    - Limit results (default 50)
    - Return list of dicts with: id, platform, filename, inserted_count, updated_count, skipped_count, imported_at

    **6. GET /ingestion/bookings — Unified bookings list**
    ```python
    @router.get("/bookings")
    def list_bookings(
        platform: str | None = None,
        property_slug: str | None = None,
        limit: int = 100,
        offset: int = 0,
        db: Session = Depends(get_db),
    ):
    ```
    - Query Booking table joined with Property table (for property_slug)
    - Optional platform filter
    - Optional property_slug filter (join to Property.slug)
    - Pagination via limit/offset
    - Return list of dicts with minimum fields per CONTEXT.md: platform, confirmation_code (platform_booking_id), guest_name, check_in_date, check_out_date, net_amount, property_slug
    - Order by check_in_date descending

    **7. GET /ingestion/bank-transactions — Bank transactions list**
    ```python
    @router.get("/bank-transactions")
    def list_bank_transactions(
        limit: int = 100,
        offset: int = 0,
        db: Session = Depends(get_db),
    ):
    ```
    - Query BankTransaction table
    - Pagination via limit/offset
    - Return list of dicts with: transaction_id, date, description, amount
    - Order by date descending
