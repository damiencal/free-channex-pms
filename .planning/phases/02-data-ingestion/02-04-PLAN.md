    This function:
    1. Collects all row-level errors
    2. Normalizes amounts from "Payable To You" column (strip currency symbols, commas)

       - If single column with range (e.g., "01/15/2025 - 01/20/2025"), split on " - " and parse each

    5. Resolves VRBO Property ID to property_slug using listing_slug_map from config
    6. Returns (records, errors)

    **Date range parsing:**
    ```python
    def _parse_check_in_out(raw: str | None, row_num: int) -> tuple[date | None, date | None, str | None]:
        """Parse VRBO Check In/Check Out field. May be 'MM/DD/YYYY - MM/DD/YYYY' or similar."""
        if raw is None or raw.strip() == "":
            return None, None, f"Row {row_num}: Check In/Check Out is missing"
        parts = raw.split(" - ")
        if len(parts) != 2:
            # Try other separators
            parts = raw.split(" to ")
        if len(parts) != 2:
            return None, None, f"Row {row_num}: Check In/Check Out format not recognized ({raw!r})"
        check_in, err_in = _parse_date(parts[0].strip(), "Check In", row_num)
        check_out, err_out = _parse_date(parts[1].strip(), "Check Out", row_num)
        errors = [e for e in (err_in, err_out) if e]
        return check_in, check_out, "\n".join(errors) if errors else None
    ```

    **Payout grouping by Reservation ID:**
    - Group all rows by Reservation ID
    - For each group: sum "Payable To You" amounts (net payout), take first guest name, first Check In/Check Out, first Property ID
    - Rows with empty Reservation ID are skipped (may be summary/total rows)
    - Store original grouped rows as raw_platform_data

    **Property ID to property_slug resolution:**
    - Same pattern as Airbnb adapter: build flat dict from all properties' listing_slug_map
    - VRBO Property ID is the key to look up in listing_slug_map
    - Error message: "Row {N}: VRBO Property ID '{id}' not found in any property's listing_slug_map"

    **Amount normalization:**
    ```python
    def _normalize_amount(raw: str | None, col_name: str, row_num: int) -> tuple[Decimal | None, str | None]:
        if raw is None or raw.strip() == "":
            return Decimal("0"), None  # VRBO may have empty amounts for some payment types
        cleaned = raw.strip().replace("$", "").replace(",", "").replace("(", "-").replace(")", "")
        try:
            return Decimal(cleaned), None
        except Exception:
            return None, f"Row {row_num}: {col_name} is not a number ({raw!r})"
    ```
