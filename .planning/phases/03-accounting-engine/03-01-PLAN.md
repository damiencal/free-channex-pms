**app/models/journal_entry.py:**
- `JournalEntry` model on table `journal_entries`
- Fields: `id` (int PK), `entry_date` (Date, not null), `description` (String(512), not null), `source_type` (String(64), not null -- e.g. 'booking_payout', 'expense', 'loan_payment', 'reversal', 'adjustment'), `source_id` (String(256), unique, not null -- idempotency key), `property_id` (Integer, FK to properties.id, nullable -- None for shared entries), `created_at` (DateTime(timezone=True), server_default=func.now())
- Add `lines` relationship to JournalLine with cascade="all, delete-orphan"

**app/models/journal_line.py:**
- `JournalLine` model on table `journal_lines`
- Fields: `id` (int PK), `entry_id` (int, FK to journal_entries.id, not null, indexed), `account_id` (int, FK to accounts.id, not null, indexed), `amount` (Numeric(12, 2), not null -- positive=debit, negative=credit), `description` (String(255), nullable)
- Add `entry` relationship back to JournalEntry, `account` relationship to Account

**app/models/__init__.py:**
- Add imports for Account, JournalEntry, JournalLine (follow existing pattern of importing all models)

**app/accounting/__init__.py:**
- Empty init file (just a docstring)

**app/accounting/journal.py:**
- Define `LineSpec` dataclass: `account_id: int`, `amount: Decimal`, `description: str | None = None`
- Define `create_journal_entry(db: Session, entry_date: date, description: str, source_type: str, source_id: str, lines: list[LineSpec], property_id: int | None = None) -> JournalEntry | None`
- Balance enforcement FIRST: `sum(line.amount for line in lines)` must equal `Decimal("0")` -- raise ValueError if not
- Minimum 2 lines -- raise ValueError if fewer
- Use `pg_insert(JournalEntry).values(...).on_conflict_do_nothing(index_elements=["source_id"])` for idempotent insert
- If rowcount == 0, return None (already exists)
- Otherwise fetch inserted entry via `db.query(JournalEntry).filter_by(source_id=source_id).one()` to get ID
- Create JournalLine rows for each LineSpec
- Call `db.flush()` (caller commits)
- Return the JournalEntry

IMPORTANT: Use `from decimal import Decimal` everywhere. Never use float for monetary amounts. Use `Numeric(12, 2)` (not 10,2 -- Phase 3 uses 12,2 per research). Import `from sqlalchemy.dialects.postgresql import insert as pg_insert`.
